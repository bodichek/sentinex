Postup nastavení (krok za krokem)
  1. V Google Cloud Console vytvoř Service Account a stáhni JSON klíč.
  2. Povol API: Gmail, Drive, Calendar, Docs, Sheets, Slides, Admin SDK, People.
  3. V Workspace Admin Console → Security → API Controls → Domain-Wide Delegation: přidej Unique ID této SA a vlož všechny scopes (jsou níže).
  4. Doplň GOOGLE_WORKSPACE_SA_JSON_PATH, GOOGLE_WORKSPACE_DOMAIN, GOOGLE_WORKSPACE_ADMIN_EMAIL v .env.
  5. Restartuj Sentinex.
